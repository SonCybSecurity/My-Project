# My-Project
Software used: Nessus, Wireshark, HxD.

Computer Network Security: To evaluate Nessus's vulnerability scanning capabilities, our team set up a black-box test in a real-world network environment. We are going to use Tenable Nessus Essentials for scan and update the number of Plugins used.

Digital Forensics: The core software tools selected include Wireshark version 4.x, serving as the packet sniffer and analyzer, allowing detailed observation of each data frame transmitted over the network.  HxD software is utilized as a Hex Editor, serving the process of raw data processing and file recovery.
